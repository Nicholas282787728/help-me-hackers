wordpressの定番プラグイン「Breadcrumb NavXT」 の改良

Retweetをすることで、このタスクにGood!ポイントが加算されます

GPLv2
ライセンス

wordpressの定番になっているパンくずリストの自動生成プラグイン、「Breadcrumb NavXT」の改良をお願い致します。

http://mtekk.weblogs.us/code/breadcrumb-navxt/

問題点

  1. Breadcrumb NavXTには表示させる文字数を制限する機能が付いているが、その機能がマルチバイトに対応していない為、文字化けしてしまう。
  2. Breadcrumb NavXTが出力するhtmlがあまり好きではない。

改良していただきたいこと。

  1. Breadcrumb NavXTには表示させる文字数を制限したときの文字化の解消。
  2. Breadcrumb NavXTが出力するHTMLの改良

Breadcrumb NavXTの吐き出すHTMLの改良について

<!-- Breadcrumb NavXT 3.4.1 -->
<a href="http://fjord.jp" title="homeへ">home</a>»<a href="http://fjord.jp/category/love" title="blogのトップページ">blog</a>»<span class="carrent">2010年4月20日、初台に引越し…</span>

…なのですが、これを

<!-- Breadcrumb NavXT 3.4.1 -->
<ol>
<li><a href="http://fjord.jp" title="homeへ">home</a>»</li>
<li><a href="http://fjord.jp/category/love" title="blogのトップページ">blog</a>»</li>
<li class="carrent">2010年4月20日、初台に引越し…</li>
</ol>

もしくは、

<!-- Breadcrumb NavXT 3.4.1 -->
<ol>
<li><a href="http://fjord.jp" title="homeへ">home</a>»</li>
<li><a href="http://fjord.jp/category/love" title="blogのトップページ">blog</a>»</li>
<li><span class="carrent">2010年4月20日、初台に引越し…</span></li>
</ol>

と、表示させられるようにしていただきたいです。

また、この矢印は、

»

管理画面で表示、非表示が出来る機能があるのですが、その機能は改良後も使えるようにしていただきたいです。

ハッカーの皆さま、宜しくお願いします。
